Компания ISsoft приглашает студентов на тренинг по JavaScript!

Описание тренинга:

Тренинг для JavaScript - разработчиков программного обеспечения. Он предназначен для студентов, которые имеют навыки программирования и базовые знания ООП.

Обучение во время первого этапа направлено на изучение основ HTML, CSS и JavaScript с ориентацией на разработку JavaScript.

Второй этап тренинга является практическое углубленное изучение веб-технологий.

Данный тренинг посвящен актуальным подходам, технологиям и инструментальным средствам разработки на Javascript. Посещая занятия данного курса, вы узнаете о последних тенденциях и трендах в JavaScript (с фокусом на ООП), познакомитесь с HTML, CSS и HTTP; научитесь выполнять юнит-тестирование клиентского кода; познакомитесь с архитектурой приложений и структурой типовых проектов, а также получите представление об инструментах современного Javascript-разработчика.

Необходимые знания и умения

  • техническое образование
  • уровень письменного и разговорного английского – не ниже среднего
  • аналитические способности и сообразительность при решении нестандартных задач
  • уверенные знания алгоритмов и структур данных, ООП, баз данных
  • опыт программирования на любом из языков
  • базовые знания JavaScript, HTML/HTML5, CSS/CSS3
